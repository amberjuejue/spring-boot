```

# 如果希望对文字美化，可以使用一个在线工具：http://patorjk.com/software/taag，本例文本内容为：ZLIKUN.COM
# 默认Banner文本：
  .   ____          _            __ _ _
 /\\ / ___'_ __ _ _(_)_ __  __ _ \ \ \ \
( ( )\___ | '_ | '_| | '_ \/ _` | \ \ \ \

 :: Spring Boot ::        (v1.5.4.RELEASE)
```
